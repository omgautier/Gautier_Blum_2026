# Gautier, Blum et al. 2026 Code Repository
This repository contains code associated with Gautier, Blum et al., “An emergent disease-associated motor neuron state precedes cell death in ALS,” including snRNA-seq, snATAC-seq/multiome, MERFISH, and human ALS analyses.

## Data availability
- snRNA-seq: [GEO: GSE306676](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE306676)
- Paired snRNA-seq/snATAC-seq multiome: [GEO: GSE306675](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE306675)
- Processed MERFISH spatial transcriptomics data: [Zenodo: 10.5281/zenodo.16938739](https://doi.org/10.5281/zenodo.16938739)
- Interactive web portal: [spinalcordatlas.org](http://www.spinalcordatlas.org)
